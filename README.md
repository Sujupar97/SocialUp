# SocialUp 🚀

Content Management & Distribution Platform for Creators

![SocialUp Logo](public/logo.png)

## Features

- 📹 **Video Management** - Upload/organize content
- 🔄 **Multi-Platform Distribution** - TikTok, Instagram & more
- 🤖 **AI Descriptions** - Auto-generated captions
- 📊 **Analytics Dashboard** - Track performance
- ⚡ **Automation** - Schedule & auto-publish

## Tech Stack

- React + TypeScript + Vite
- Framer Motion
- Supabase
- N8N (Automation)

## Quick Start

```bash
npm install
npm run dev
```

## Deploy

This project is configured for Netlify:

```bash
npm run build
```
